Синхронизация с GitHub

Токен хранится только в памяти вкладки (не сохраняется и никуда не отправляется, кроме api.github.com). Но т.к. запрос идёт напрямую из браузера, любой, кто откроет консоль разработчика на этой странице в момент сохранения, теоретически может его увидеть. Для публичного сайта надёжнее делать запись через маленький серверный прокси (Cloudflare Worker / Netlify Function), а не хранить токен в браузере.
